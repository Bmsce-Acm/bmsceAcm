# BMSCE ACM

## To start this project run,
- For Python setup:
```
pip install pipenv
pipenv shell
pipenv install -r requirements.txt
```

- For Node setup:
```
cd .\bmsceAcm\frontend
npm i
npm run build
```

- For any changes in frontend to appear run this after the changes in frontend
```
npm run build
```

- To start the App
```
cd .\bmsceAcm\
python manage.py runserver
```
